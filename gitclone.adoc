== Getting started

Run the following command to clone the Git repository and use the projects provided
inside:

[subs="attributes"]
----
git clone https://github.com/openliberty/guide-{projectid}.git
cd guide-{projectid}
----

The `start` directory contains the starting project that you can use to begin.

The `finish` directory contains the finished project. To see what you'll build, run the commands in this directory.
